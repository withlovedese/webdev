Working with Statements in Javascript

If Else Statements: checks if a condition is true and proceeds to execute the code within the if block but if false does not execute

If Else Statements: checks if a condition is true and proceeds to execute the code within the if block but if its false executes the code block within the else block

If Else if Statements: checks if a condition is true and proceeds to execute the code within the if block but if its false executes the code block within the else block

Nested if Statements: an if statement that containes one or more if statements.

Switch Case Statements: used to simplify code to avoid multiple if statements.
