Arithmetic operators

  • Addition (+)
  • Subtraction (-)
  • Multiplication (*)
  • Divison (/)
  • Modulo (%)

Relational operators

Equal to (==)

Checks if the two values are the same

Not Equal to (!=)

Checks if the two values are not the same

StrictEqual to (===)

Checks if the two values are the same AND if the data type is the same

Greater than (>)

Checks if first value is greater than second value

Greater than or equal to (>=)

Checks if first value is greater than or equal to second value

Less than (<)

Checks if first value is less than second value

Less than (<=)

Checks if first value is less than or equal to second value

Logical Operators

NOT operators (!): Returns true if operation is false and vice versa

OR operators (||): Returns true if either first or second operation is true else returns false

AND operators (&&): Returns true if both first and second operations are true else returns false and

Assignment Operators

Assignment operator (=): assigns value on right side to the left side

Compound Assignment operator (+=): updates the value on left side by adding value on right side to original left side value
